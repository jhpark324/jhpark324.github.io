## 리눅스 기본 명령어

###### cd(Change Directory) : 

- cd <폴더이름> -> 폴더 이동 
- cd .. -> 상위 폴더 이동

---

###### pwd(Print Working Directory) :

- pwd -> 현재 경로 프린트
- 루트 유저가 아닌이상 모두 home 디렉토리

---

###### ls(List) :

- ls -> 현재 폴더에 있는 모든 파일 보기
- ls -l -> 자세히 보기
- ls -a -> 숨긴 파일 포함 
- ls -h 크기를 표시

---

###### whoami:

- 내가 어떤 사용자인지 출력

---

###### hostname:

- 현재 접속중인 호스트 이름 출력

---

 ###### ip addr(ifconfig):

- 랜카드와 ip 정보 확인 

- lo(Loopback 인터페이스): 자기 자신을 가리키는 네트워크
  - mtu: 한 번에 전송 가능한 패킷의 최대 크기(Byte)
  - inet: IPv4 주소
  - inet6: IPv6 주소
- ens33(네트워크 카드 이름): 물리 네트워크:
  - <BROADCAST,MULTICAST,UP,LOWER_UP>: 브로드캐스트 가능, 멀티캐스트 가능, 인터페이스 켜져있음, 실제 물리 연결됨
  - mtu 1500: 이더넷 패킷 최대S 크기 
  - link/ether: 실제 MAC주소
  - inet: IPv4 주소
  - inet6: IPv6 주소
  - brd: 같은 네트워크에 있는 모든 장치로 보내는 주소
  - scope global dynamic: DHCP로 받은 글로벌 IPv4

---

###### sudo su:

- root유저로 전환
- exit를 통하여 탈출

---

###### sudo:

- root유저로 전환 없이 sudo 권한 사용
-  `sudo -l` : 현재 사용가능한 범위 확인

---

##### 파일 디렉토리

###### /boot

- 부팅관련 파일

###### /dev

- 디바이스 관련 파일 존재 (ex: 하드 드라이버, usb 디바이스)

###### /etc *

- 시스템에 관련된 환경파일 (ex: hostname, sudoers)

###### /bin *

- 바이너리 파일(컴파일된 파일) (ex: ls)

###### /sbin

- 시스템에 관련된 바이너리 파일 (ex: ip, fdisk)

###### /opt *

- 추가적인 소프트웨어 파일 (외부 소프트웨어)

###### /proc

- 프로세스와 실행중인 시스템 정보 파일 (ex: 시스템 메모리, 디바이스)

###### /lib

- 라이브러리 파일
- 커널에 대한 유틸리티 파일

###### /tmp

- 프로그램들이 잠깐 저장하는 임시파일들
- 시스템적으로 주기적으로 비우게 해야함

###### /home *

- root유저가 아닌 외부 유저들의 디렉토리

###### /var *

- 변하는 데이터가 저장되는 파일 (ex: 로그파일)

###### /run

- 시스템이 부팅된 이후에 생기는 실행 중 정보를 저장하는 파일 (ex: 사용자별 세션 정보, 프로세스 락 파일 등)

- /proc는 (실시간 상태) 와 /var (저장되는 데이터)의 중간 개념

###### /mnt 

- 하드디스크나 네트워크 드라이브 같은 외부 저장장치를 임시로 연결할때 사용하는 파일

###### /media

- 외부장치가 자동으로 연결되는 위치 (데스크탑 환경)
- /media/사용자이름/usb이름

###### /usr

- 사용자용 응용 프로그램, 라이브러리, 문서 저장된 폴더

##### 파일관리 

###### touch

- 빈 파일을 만들때 사용

###### cp 

- 파일이나 디렉토리를 복사할때 사용
- ex) `cp a.txt b.txt` a를 복사해서 b파일만듬
- ex) `cp a.txt /home/user/Documents/` a.txt를  경로안으로 복사
- ex) `cp -r myfolder backup_folder` myfolder 안에 있는 모든 파일을 backup__folder로 복사

###### vi(/vim)

- 파일을 편집할때 많이 사용

###### mkdir 

- 폴더를 만들때 사용
- -p 로 경로로 지정가능

###### rm

- 파일을 지울때 사용
- -r로 폴더를 지울 수 있음

- -rf로 묻지않고 삭제 가능

###### find

- 파일을 찾을때 사용
- find <검색경로> <검색조건> <실행할명령>
- ex) `find /home -name 'a.txt'` 
- ex)` find .  -name "*.txt"` .현재 경로, *.txt txt확장자 검색

###### locate 

- 파일 목록 db를 만들어서 경로를 찾아줌

###### whatis

- 명령어나 프로그램이 무엇을하는지 설명해줌

###### --help

- 옵션들을 볼 수 있음

###### man

- 명령어의 자세한 설명을 보여줌

---

##### vim

###### gg

- 맨 위 라인으로 감

###### G

- 맨 아래 라인으로 감

###### ^

- 라인의 맨 앞으로 감

###### $

- 라인의 맨 뒤로 감

###### :<number>

- 해당 라인으로 이동

###### i

- 에디터모드로 변경

###### x or X

- 에디터 모드 진입 없이 현재 커서에 기준으로 값 하나 삭제

###### dd

- 에디터 모드 진입 없이 현재의 라인 삭제

###### yy

- 현재 라인 복사

###### p

- 붙여넣기

###### /<text> 

- 값 찾기

###### :%s/old/new/g

- 특정 값을 새로운 값으로 변경

###### u

- 돌아가기

###### :w

- 저장

###### :q

- 탈출

###### :wq or ZZ

- 저장후 탈출

###### :q!

- 저장 없이 나감

###### :set number

- 디스플레이에 라인을 숫자로 보여줌

##### crontab

- 정해진 시간마다 명령어나 스크립트를 자동으로 실행하게 설정하는 시스템
- 리눅스의 예약 작업
- -e: 현재 사용자의 crontab 파일을 편집기로 열어서 예약 작접을 추가 or 수정
- -l: 현재 사용자에 설정된 모든 예약 작업 목록으로 출력
- -r: 현재 사용자에 등록된 모든 예약 작업을 삭제
- -u <user>: 특정 사용자의 예약 작업을 관리 (root 권한)
