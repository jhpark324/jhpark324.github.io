## 리눅스 기본 명령어

###### cd(Change Directory) : 

cd <폴더이름> -> 폴더 이동 <br> 
cd .. -> 상위 폴더 이동



###### pwd(Print Working Directory) :

pwd -> 현재 경로 프린트



###### ls(List) :

ls -> 현재 폴더에 있는 모든 파일 보기<br>ls -l -> 자세히 보기<br>
ls -a -> 숨긴 파일 포함 <br>
ls -h 크기를 표시
